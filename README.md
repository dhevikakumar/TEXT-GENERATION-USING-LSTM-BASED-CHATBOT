This project focuses on developing a simple chatbot using an LSTM-based sequence-to-sequence model.
The chatbot is trained on a small dataset of question and answer pairs to understand basic conversational patterns. 
Text data is preprocessed using tokenization and padding before being fed into the model. The encoder processes the
input sentence, and the decoder generates a relevant response word by word. After training, the chatbot can produce 
meaningful replies to user queries, demonstrating basic natural language processing and text generation capabilities.
